# @node-cron/nestjs

A drop-in replacement for [`@nestjs/schedule`](https://docs.nestjs.com/techniques/task-scheduling)
backed by [node-cron](https://github.com/merencia/node-cron): same decorators,
plus distributed scheduling, background tasks and per-fire coordination.

> 🚧 Initial project scaffolding. The scheduler plugin lands via the first pull
> request.
